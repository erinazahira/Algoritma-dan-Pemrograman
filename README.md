# Program Tabungan Sederhana
Program tabungan sederhana merupakan program yang bertujuan untuk membantu mencatat alokasi dana tabungan. Dalam program berikut, tersedia tiga pilihan tabungan yang dapat digunakan pengguna. Tiga jenis tabungan tersebut diantaranya adalah tabungan darurat, tabungan biasa, dan tabungan pendidikan.
# Fitur Tabungan Sederhana
* Menambah atau memasukkan dana pada tabungan.
* Mengecek rincian saldo yang terdapat dalam tabungan.
* Menarik atau mengambil saldo dari tabungan.
* Menampilkan data pemasukkan dan pengeluaran tabungan.
* Menampilkan pie chart saldo dari masing-masing jenis tabungan.
* Menampilkan bar plot pemasukkan dan pengeluaran.
